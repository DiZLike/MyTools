Icon Finder — Локальный поисковик иконок

Версия: 1.0
Год выпуска: 2026
Платформа: Windows 10/11 (x64)
Язык интерфейса: Русский

Описание:
IconFinder — это мощный оффлайн-инструмент для поиска, просмотра и конвертации иконок (более 100 000 иконок в форматах SVG, PNG, ICO, WebP). Программа позволяет быстро находить нужные иконки по ключевым словам на русском или английском языке, конвертировать их в различные форматы и размеры, а также перетаскивать иконки напрямую в другие приложения.

Особенности программы:

✅ Огромная база иконок — более 100 000 иконок в различных стилях (iOS, Material Design, Windows 10 и др.)
✅ Умный поиск — поддержка поиска на русском языке с автоматическим AI-переводом на английский
✅ Конвертация форматов — экспорт в PNG, ICO, WebP с выбором размера (16×16 до 512×512)
✅ Drag & Drop — перетаскивание иконок прямо из программы в другие приложения
✅ Современный интерфейс — тёмная тема, адаптивный дизайн, бесконечная прокрутка
✅ Ленивая загрузка — миниатюры подгружаются по мере прокрутки, не нагружая память
✅ Полный оффлайн — не требует интернета для работы
✅ Portable — не требует установки, можно запускать с флешки

Поддерживаемые форматы:

Исходные: SVG (векторные), PNG (растровые)
Экспорт: PNG, ICO, WebP

Системные требования:

Windows 10/11 (64-bit)
.NET 10.0 Runtime (включён в раздачу)
~2 ГБ свободного места на диске
Для AI-перевода требуется Python (включён в раздачу)

Состав раздачи:

IconFinder.exe — основной исполняемый файл
Папка Data — база данных иконок (icons.db + icons.dat)
Папка Data/Python — встроенный Python для AI-перевода
Папка Data/ru-en.dict — словарь русско-английского перевода
Дополнительные библиотеки .NET

Инструкция по запуску:

Скачать раздачу
Запустить IconFinder.exe
При первом запуске программа предложит распаковать компоненты для перевода (выполняется один раз)
Готово! Можно искать иконки

Советы:

Введите 2+ символов для поиска
Нажмите на иконку для подробной информации
Кнопка "💾 Сохранить" — экспорт в выбранном формате
Drag & Drop — перетащите иконку в любое приложение
Прокрутка вниз — автоматическая подгрузка новых иконок
